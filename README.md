# Student Performance Predictor

## Overview
This Machine Learning project predicts student final grades (G3) using Linear Regression.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Dataset
Student Performance Dataset containing demographic, academic, and social attributes of students.

## Features Used
- Student demographic information
- Family information
- Study time
- Previous grades (G1, G2)
- Attendance
- Social and lifestyle factors

## Model
Linear Regression

## Results
- R² Score: 0.724
- Mean Absolute Error (MAE): 1.65

The model explains approximately 72.4% of the variance in student final grades.
